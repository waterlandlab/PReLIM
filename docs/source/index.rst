======================================================
PReLIM: Precise Read-Level Imputation of Methylation
======================================================


What is PReLIM?
===================================
CluBCpG is a python package that uses machine learning to imputes missing CpG methylation data at the read-level. PReLIM is integrated into CluBCpG 
for easy use.
